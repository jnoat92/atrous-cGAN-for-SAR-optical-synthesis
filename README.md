

# atrous-cGAN-for-SAR-optical-synthesis

Implementation of the atrous cGAN and pix2pix for the SAR-optical translation oriented to tow applications: crop recognition and deforestation detection.

## Dependencies

- Python 3.7
- Numpy 1.16.1
- Tensorflow 1.14.1
